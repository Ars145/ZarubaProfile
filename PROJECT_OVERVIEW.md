# ZARUBA Clan Management - Полный Обзор Проекта

## 🎮 О Проекте

**ZARUBA** - это веб-платформа для управления игровыми кланами с интеграцией статистики из игры Squad. Проект сочетает современный фронтенд с мощным бэкендом для полного управления игровым сообществом.

---

## 🎨 Frontend - Дизайн и Функционал

### Дизайн-Система

#### Цветовая Палитра
```css
/* Основные цвета */
--primary: #FF6600 (Оранжевый ZARUBA)
--background: #0a0a0a (Глубокий черный)
--foreground: #ffffff (Белый текст)

/* Темы кланов */
- Orange (Alpha Squad) - #FF6600
- Blue (Delta Force) - #3B82F6
- Yellow (Zaruba Elite) - #EAB308
```

#### Типографика
- **Display Font**: Oxanium (заголовки, uppercase, tracking-wide)
- **Body Font**: Inter (основной текст, антиалиасинг)
- **Mono Font**: System monospace (код, технические данные)

#### Визуальные Эффекты
1. **Фоновый слой**:
   - Тонкая сетка (60x60px, opacity 2%)
   - Радиальный градиент виньетки
   - Система частиц (opacity 40%)

2. **Интерактивные элементы**:
   - Magnetic buttons (притягиваются к курсору)
   - Tilt cards (3D наклон при наведении)
   - Glitch text (киберпанк эффект)
   - Smooth animations (Framer Motion)

3. **Glassmorphism**:
   - Полупрозрачные карточки
   - Backdrop blur
   - Gradient overlays
   - Glowing borders

## 🎭 Анимации и Эффекты

### Framer Motion Анимации

1. **Stagger Children**
   - Карточки появляются последовательно
   - Delay: 0.1s между элементами
   - Spring animation (stiffness: 100, damping: 15)

2. **Hover Effects**
   - Scale: 1.02
   - Shadow: увеличивается
   - Border glow: усиливается
   - Duration: 300ms

3. **Modal Animations**
   - zoom-in-95 (масштаб от 95% до 100%)
   - fade-in (прозрачность 0 → 1)
   - slide-in (снизу вверх)

4. **Glitch Text**
   - Offset clones (красный + синий)
   - Clip-path: random slices
   - Opacity: 0 → 0.7 on hover
   - Animation: pulse

5. **Magnetic Buttons**
   - Follow cursor (strength: 0.3)
   - Spring animation
   - Return to center on mouse leave

6. **Tilt Cards**
   - 3D rotation on mouse move
   - rotateX: ±10deg
   - rotateY: ±10deg
   - Transform: translateZ(30px)

### Интеграция с MongoDB

  // Боевая статистика
  kills: 1245,

  kd: 1.4,

  teamkills: 12,

  // Матчи
  matches: { matches: 178, won: 88, winrate: 49.44 },

  // Время (форматированное)
  playtime: "342ч",
  squadLeaderTime: "4д 7ч",
  commanderTime: "21ч",
  driverTime: "15ч",
  pilotTime: "30м",

  // Топ показатели
  topWeapon: { name: "M4A1", kills: 245 },
  topRole: { name: "Rifleman", time: "90ч 20м" },

  // Ранг
  rank: {
    current: { iconUrl: "...", score: 15420 },
    next: { iconUrl: "...", needScore: 20000 },
    progress: 77.1,
    isMaxRank: false

  // Детальная статистика

### Расчет Статистики

**Файл: `client/src/lib/statsCalculations.js`**

Функции:
- `calcVehicleTime(possess)` - время на технике
- `calcVehicleKills(weapons)` - убийства на технике
- `calcKnifeKills(weapons)` - убийства ножом
- `formatTime(time, field)` - форматирование времени
- `getTopRole(roles)` - топ роль
- `getTopWeapon(weapons)` - топ оружие
- `getCurrentRank(user, config)` - ранг и прогресс

## 🎮 Пользовательские Роли

### 3 Типа Пользователей

#### 1. **GUEST** (Гость)
- ❌ Не в клане
- ✅ Может просматривать кланы
- ✅ Может подавать заявки
- ❌ Не видит состав кланов
- ❌ Нет доступа к настройкам

#### 2. **MEMBER** (Член клана)
- ✅ В клане
- ✅ Видит состав своего клана
- ✅ Видит статистику членов
- ✅ Может выйти из клана
- ❌ Не может управлять членами
- ❌ Нет доступа к настройкам
- ❌ Не видит заявки

#### 3. **OWNER** (Владелец клана)
- ✅ Все права MEMBER +
- ✅ Управление членами (повышение/понижение)
- ✅ Удаление членов
- ✅ Просмотр заявок
- ✅ Принятие/отклонение заявок
- ✅ Настройки клана (баннер, лого, тема, требования)
- ✅ Редактирование Discord ссылки
