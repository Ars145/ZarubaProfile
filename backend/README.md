# Backend (Flask API)

Здесь будет Flask бэкенд приложения.

## Планируемая структура:

```
backend/
├── backend.py           # Точка входа Flask
├── config.py            # Конфигурация
├── requirements.txt     # Python зависимости
├── data/                # SQLite база данных
│   └── news.db         # База данных новостей
└── routes/              # API маршруты
    ├── auth.py          # Аутентификация
    ├── clans.py         # Управление кланами
    └── stats.py         # Статистика игроков
```

## Функционал (в разработке):

- Аутентификация пользователей
- CRUD операции для кланов
- API статистики (интеграция с MongoDB)
- Управление профилями игроков
- Система заявок в кланы
