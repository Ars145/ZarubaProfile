# Static Files

Папка для загружаемого пользовательского контента.

## Структура:

```
static/
├── uploads/             # Загруженные файлы
│   ├── avatars/        # Аватары пользователей
│   ├── clan-logos/     # Логотипы кланов
│   └── clan-banners/   # Баннеры кланов
└── README.md
```

## Примечания:

- Все загруженные файлы должны храниться здесь
- Файлы доступны через Nginx прокси
- Максимальный размер файла: 5MB
- Поддерживаемые форматы: JPG, PNG, WebP
